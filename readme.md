## 安装依赖
	npm install
## 本地开发
	npm run dev
	npm run server
## 生产部署
	npm run build
	npm start

## 目录解析
```
	app/---------------------------服务根目录
	--controllers/-----------------控制器目录
	--service/---------------------服务目录
	--static/----------------------打包后的静态资源目录
	--views/-----------------------打包后的模板文件
	--router.ts--------------------路由
	--app.ts-----------------------服务入口文件
	src/---------------------------页面脚本js和scss源码
	template/----------------------模板文件目录
```
